## Color Usage Guidelines

### Web Design

**Light Backgrounds:**
- Primary: Core White (`#FBFFF6`)
- Subtle green wash: Pale Mint (`#E3F7E3`)
- Cards: Whisper Gray (`#EFEFF3`)

**Dark Backgrounds:**
- Primary: Core Midnight (`#1C1E35`)
- Variation: Navy Vintage (`#2D3E6B`)

**Text on Light Backgrounds:**
- Body text: Core Midnight (`#1C1E35`)
- Secondary text: Charcoal (`#3D3D52`)
- Disabled text: Slate Gray (`#6B6B7C`)

**Text on Dark Backgrounds:**
- Body text: Core White (`#FBFFF6`)
- Secondary text: Whisper Gray (`#EFEFF3`)

**Accent Colors:**
- Primary actions: Core Green (`#2FBF39`)
- Links: Periwinkle Mid (`#8BA0D6`)
- Warnings/alerts: Core Red (`#CA1E08`) - USE SPARINGLY

**Borders:**
- Strong borders: Charcoal (`#3D3D52`)
- Subtle borders: Dove Gray (`#D4D4DC`)

---

### Scientific Plotting

**High Contrast Series (for multi-line plots):**
1. Core Red (`#CA1E08`)
2. Core Green (`#2FBF39`)
3. Navy Vintage (`#2D3E6B`)
4. Mustard (`#E6B800`)
5. Plum Vintage (`#8B5A8A`)

**Sequential Scales:**
Use tints to shades within each color family. Examples:
- Green scale: Pale Mint → Mint Tint → Spring Light → Core Green → Fern Shade → Forest Deep
- Blue scale: Powder Blue → Core Periwinkle → Periwinkle Mid → Slate Blue → Navy Vintage

**Diverging Scales:**
- Cool to warm: Blue family → Neutral grays → Red family
- Example: Navy Vintage → Slate Blue → Dove Gray → Coral Red → Core Red

**Backgrounds:**
- Use Core White (`#FBFFF6`) or very light tints for maximum contrast
- Avoid dark backgrounds in scientific plots

---

### Cartoon Illustrations

**Skin Tones:**
- Light: Sand Beige (`#D4A574`)
- Medium: Clay Mid (`#A0785A`)
- Warm: Salmon Light (`#F19B8F`)

**Vegetation:**
- Full green family from Forest Deep (`#1A5C1E`) to Pale Mint (`#E3F7E3`)
- Use darker shades for foreground, lighter tints for background

**Sky & Water:**
- Periwinkle family and blue tones
- Gradient from Powder Blue to Navy Vintage for depth

**Shadows:**
- Use darker shades within each color family
- Universal shadow: Charcoal (`#3D3D52`)

**Highlights:**
- Bright: Core Yellow (`#FFFF77`)
- Soft: Cream Light (`#FFFFBB`)
- Use lightest tints from each family

---

### Vintage Biological Aesthetic

**Green Wash Effect:**
- Apply Pale Mint (`#E3F7E3`) or Mint Tint (`#B8EDB9`) at 30-50% opacity over backgrounds
- Creates subtle biological feel without overwhelming content

**Parchment/Aged Paper:**
- Use Parchment (`#E5D4D1`) or Cream Light (`#FFFFBB`)
- Combine with Soil Brown borders for vintage document look

**Botanical Drawings:**
- Line work: Core Midnight (`#1C1E35`)
- Plant colors: Forest Deep, Fern Shade, Soil Brown
- Highlights: Spring Light, Sand Beige
- Background: Core White with Pale Mint wash

**Scientific Diagrams:**
- Labels: Navy Vintage (`#2D3E6B`)
- Emphasis: Core Red (`#CA1E08`) - sparingly
- Organic subjects: Earthtone family
- Line work: Core Midnight or Charcoal

## High Contrast Combinations (AAA Accessibility)

The following combinations meet WCAG AAA standards for contrast:

- Core Red (`#CA1E08`) on Core White (`#FBFFF6`)
- Core Green (`#2FBF39`) on Core Midnight (`#1C1E35`)
- Core Midnight (`#1C1E35`) on Core White (`#FBFFF6`)
- Core White (`#FBFFF6`) on Core Midnight (`#1C1E35`)
- Navy Vintage (`#2D3E6B`) on Core White (`#FBFFF6`)
- Forest Deep (`#1A5C1E`) on Pale Mint (`#E3F7E3`)

## Color Design Principles

1. **Sparing use of red**: Core Red should be reserved for warnings and critical highlights only. It is not for general CTAs or decorative purposes.

2. **Vintage biological feel**: Incorporate subtle green washes (Pale Mint, Mint Tint) to reinforce the organic, biological brand identity.

3. **High contrast**: Prioritize readability and clarity, especially in scientific contexts. Always test color combinations for accessibility.

4. **Earthtone grounding**: Use browns, beiges, and warm neutrals to create vintage, natural aesthetic that feels scientific and trustworthy.

5. **Blue-tinted darks**: Core Midnight has a subtle blue tint that complements the periwinkle family and adds sophistication to dark backgrounds.

## Generated Word documents (.docx)

For proposals, invoices, contracts, memos, and any other `.docx` output formatted for Word or Google Docs:

- **No horizontal rules** as section separators: avoid paragraph borders under titles, decorative dividers, and horizontal lines between sections. Use spacing and heading hierarchy instead, unless a line is strictly necessary (for example, borders inside a data table).
- **No italics** except when strictly necessary (for example, binomial nomenclature or a citation convention that requires italics). Prefer semibold weight, headings, or plain phrasing for emphasis.

## Inter (Body Text)

Inter is a highly legible sans-serif typeface designed for screens. Use it for body copy, preview excerpts, and any longer-form reading content.

**Source:** [rsms.me/inter](https://rsms.me/inter) • [GitHub](https://github.com/rsms/inter)  
**License:** OFL 1.1 (open source)

### Files (styleguide/fonts/inter/)

| File | Weight | Use |
|------|--------|-----|
| `inter-latin-400-normal.woff2` | 400 (Regular) | Body text |
| `inter-latin-500-normal.woff2` | 500 (Medium) | Emphasis |
| `inter-latin-700-normal.woff2` | 700 (Bold) | Strong emphasis |

### @font-face Declaration

```css
@font-face {
  font-family: 'Inter';
```
